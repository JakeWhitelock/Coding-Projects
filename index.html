Brief Information

Entering "N/A" in both the Emblem shape and Defining feature will return all data within the database, provided they are the only two fields that are used.

The "Lettering" option is designed to allow you to enter any abbreviations or bold lettering that may be present in the school emblem.

The check boxes can be used as a wild card in order to get information on all clothing of that type and can be used in combination with other search options if you have other clothing types to search for.

The clothing text boxes can be used on their own, to specify a known colour for that article of clothing. This gives a specific response for any matches and can be used in combination with other fields for a more narrow set of results.
